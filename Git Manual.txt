1. Завести аккаунт на GitHub

2. Создать репозиторий

3. По инструкции:

- Создать папку, где будет храниться проект

- Зайти в нее и инициализировать Git: 

#git init

- Создать или переписать файлы проекта и добавить их в систему учета Git:

#git add *

- Создать ПЕРВЫЙ комментарий по проекту: 

#git commit -m "Project Start"

- Далее указать путь к удаленному репозиторию

#git remote add origin git@github.com:torrison/sandbox.git

- И отправить данный на сервер (Хаб)

#git push -u origin master

(По идее, предварительно, нужно создать SSH ключ, для безопастной передачи данных)

- Теперь добавляем еще один файл или группу файлов в проект
